# Database
## 팀 관리 데이터베이스
요구사항 정리
1. 선수는 선수등록번호, 이름, 소속, 포지션, 등번호, 생년월일, 키 ,몸무게를 가진다.
2. 팀은 이름, 지역, 구장을 가진다.
3. 리그는 리그이름, 지역을 가진다.
4. 구장은 구장이름, 팀 이름, 위치를 가진다.
5. 선수는 하나의 팀에 소속된다.
6. 팀은 여러명의 선수를 포함한다.
7. 팀은 하나의 구장을 관리한다.
8. 구장은 하나의 팀에 관리받는다.
9. 리그는 여러팀이 참가한다.
10. 팀은 여러 리그에 참가한다.
11. 리그는 여러 구장을 이용한다. 
12. 구장은 여러 리그에 이용된다.
13. 선수는 한명의 주장을 따른다.
14. 주장은 여러 선수들을 리드한다.
